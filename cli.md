# CurrencyKit CLI (Scaffold)

A simple CLI tool to add or update currency symbols in your project.

## Usage

```
npx currency-kit-cli add --code=USD --symbol="$" --svg=./svgs/USD.svg
npx currency-kit-cli update --code=INR --svg=./svgs/INR.svg
```

## Features (Planned)
- Add a new currency symbol
- Update an existing currency symbol
- Validate SVG files
- List all supported currencies
- Interactive prompts
