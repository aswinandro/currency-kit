# currency-kit Project Plan

This file will track the plans, features, and todos for the currency-kit npm package.

## Goals
- Provide currency symbols and codes for all countries
- Allow customization (size, color, gradient)
- Work seamlessly in React Native on any device
- Include demo component and CLI for adding currencies

## Planned Steps
1. Project setup (npm, folder structure)
2. Core library implementation
3. Customization options (size, color, gradient)
4. Demo component
5. CLI tool
6. Documentation

---

## Advanced/Sophisticated Features (Ideas)

- Animated currency symbols (bounce, fade, spin)
- Support for symbol placement (before/after amount, RTL/LTR)
- Automatic currency detection by locale/country
- Currency flag icons alongside symbols
- Accessibility support (screen reader labels, ARIA tags)
- Currency formatting utilities (e.g., 1,000.00 vs 1.000,00)
- Theming support (dark/light mode adaptation)
- SVG and font-based symbol rendering options
- Currency conversion rates integration (optional API)
- Offline support for all symbols and codes
- Multi-language support for currency names
- Custom symbol upload/override by users
- Tree-shakable exports for smaller bundle size
- TypeScript types and auto-complete support
- Unit tests and Storybook for all components

Add more details and todos as the project progresses.
